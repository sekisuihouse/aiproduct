# Role Rotation Plan

## Weekly Round

### Session 1: Executive Control

- CEO
- COO
- CFO
- CTO
- CPO
- CMO
- CRO
- CSO
- Chief of Staff
- Business Owner

Questions:

- 何を捨てるか
- 何を伸ばすか
- 何が詰まっているか
- どの数字が悪化しているか

### Session 2: Product and Design

- PdM
- PM
- PO
- UX Designer
- UI Designer
- UX Researcher
- Service Designer
- Brand Designer
- Graphic Designer
- Copywriter

Questions:

- 顧客は本当に困っているか
- 体験は一歩目でわかるか
- どこで迷うか
- 何を削るべきか

### Session 3: Engineering and Data

- Tech Lead
- Frontend Engineer
- Backend Engineer
- Fullstack Engineer
- Mobile App Engineer
- Infrastructure Engineer
- SRE
- DevOps Engineer
- QA Engineer
- Security Engineer
- Data Engineer
- Data Analyst
- Data Scientist
- AI / ML Engineer
- Tech Support

Questions:

- どこが壊れるか
- どこが遅いか
- どこが危ないか
- 何が測れていないか

### Session 4: Marketing and PR

- Marketing Manager
- Growth Hacker
- Content Marketer
- SEO
- SNS Marketer
- Ad Operations
- PR
- Community Manager

Questions:

- どこで認知が生まれるか
- 何が紹介を生むか
- 何を発信すべきか
- 何を継続するか

### Session 5: Sales and CS

- Head of Sales
- Inside Sales
- Field Sales
- Business Development
- Customer Success

Questions:

- 誰に刺さるか
- 失注理由は何か
- 導入障壁は何か
- 継続価値は何か

### Session 6: Operations

- HR
- Legal

Questions:

- 採用と責任分界は妥当か
- 契約と規程に抜けはないか
- リスクは先に潰せるか

## Rule

毎週この順番で論点を回し、重要事項は `02_strategy/decision-log.md` に転記する。
